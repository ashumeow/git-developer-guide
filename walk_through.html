Sage Development Process¶

This section will present a concise overview of the Sage development process. In it, we will see how to obtain the Sage source code repository, make changes, and communicate these changes back to the Sage project. If you are a beginner to Sage development, this introductory guide is here to help you become familiar with the Sage development process.
